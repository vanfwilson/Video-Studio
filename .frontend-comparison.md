## Migration Strategy

### Option 1: Incremental Port (Recommended)
1. Copy shadcn/ui components from A to C
2. Port pages one by one
3. Add API endpoints as needed

### Option 2: Replace Frontend
1. Use Version A frontend as-is
2. Point to Version C API
3. Add missing API endpoints to match

### Option 3: Hybrid
1. Keep C frontend for core video features
2. Add new pages from A for business tools
3. Share common components

- **Action**: Port query hooks from A

### Dropbox Integration
- Version A: Uses Replit-specific Dropbox connector

- **Action**: Need standard Dropbox OAuth implementation
